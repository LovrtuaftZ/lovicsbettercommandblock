# Lovic's Better Command Block 1.20.1 Fabric - 编译与导出

## 1. 环境要求

- Java 17（与 Forge 1.20.1 一致）
- 与 `VicColor-1.20.1 Forge` 相同的 ForgeGradle / MixinGradle 配置（本项目已对齐）

## 2. 首次准备（可选）

当前工程已包含 `gradlew.bat` 与 `gradle/wrapper/gradle-wrapper.properties`。  
如果你本机没有 `gradle/wrapper/gradle-wrapper.jar`，可先执行：

```powershell
gradle wrapper
```

执行后会生成 `gradle-wrapper.jar`，之后就能直接使用 `.\gradlew.bat`。

## 3. 编译命令

在项目根目录执行：

```powershell
.\gradlew.bat build
```

若你的环境是直接安装了 Gradle，也可以：

```powershell
gradle build
```

## 4. 导出产物

编译成功后，Jar 文件在：

- `build/libs/lovicsbettercommandblock-1.1.1.jar`

该文件可直接放入 Minecraft `mods` 文件夹（Forge 1.20.1）。

## 5. 功能验证建议

1. 进入游戏放置命令方块并打开 GUI。
2. 在命令输入区域输入长指令，确认：
   - 输入框高度明显增大；
   - 文本自动换行（不再强制单行横向显示）；
   - 超出底部时可使用鼠标滚轮上下滚动；
   - 右侧出现滚动条并可拖动。
